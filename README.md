# Deep Reinforcement Learning Materials

This repository contains homework assignments and notebooks from the **Deep Reinforcement Learning course**.

## Contents
- Introduction to Reinforcement Learning — MDPs, exploration vs. exploitation  
-  Value‑Based Methods — Q‑Learning, Deep Q‑Networks (DQN), Double DQN  
-  Policy‑Based Methods — REINFORCE, Actor‑Critic, A2C, A3C  
-  Advanced Methods — PPO, TRPO, SAC, TD3  
-  Model‑Based Methods — World Models, Dyna‑Q, Model Predictive Control (MPC)  
-  Multi‑Armed Bandits — ε‑greedy, UCB, Thompson Sampling  
-  Value‑Based Theory — Bellman equations, convergence proofs, function approximation  
-  Policy‑Based Theory — Policy gradient theorem, variance reduction, entropy regularization  
-  Imitation & Inverse RL — Behavioral Cloning, GAIL, IRL  
-  Offline Methods — Batch RL, Conservative Q‑Learning (CQL), BRAC  

More information can be found at the course website: https://deeprlcourse.github.io/
